# remnawave_node_action (Resource)

Executes a one-time action on a Remnawave node. Supported actions:

| Action          | Backend endpoint                                    | Description                                   |
|-----------------|-----------------------------------------------------|-----------------------------------------------|
| `enable`        | `POST /api/nodes/actions/:uuid/enable`              | Re-enable a disabled node.                    |
| `disable`       | `POST /api/nodes/actions/:uuid/disable`             | Administratively disable a node.              |
| `restart`       | `POST /api/nodes/actions/:uuid/restart`             | Restart the Xray backend on the node.         |
| `reset-traffic` | `POST /api/nodes/actions/:uuid/reset-traffic`       | Reset the node's traffic counter to zero.     |

Because the action is imperative (it does not represent a long-lived object on the server), the resource re-executes the action whenever any of its arguments change. Use the `triggers` attribute to control when re-execution occurs — a common pattern is `triggers = [timestamp()]` to force the action on every `terraform apply`.

## Example Usage

```terraform
resource "remnawave_node_action" "restart" {
  node_uuid     = remnawave_node.example.uuid
  action        = "restart"
  force_restart = true
  triggers      = [timestamp()]
}
```

```terraform
resource "remnawave_node_action" "reset_traffic" {
  node_uuid = remnawave_node.example.uuid
  action    = "reset-traffic"
  triggers  = [var.traffic_reset_trigger]
}
```

## Schema

### Required

- `action` (String) Action to perform. One of: `enable`, `disable`, `restart`, `reset-traffic`.
- `node_uuid` (String) UUID of the target node.

### Optional

- `force_restart` (Boolean) Whether to force-restart the node (only meaningful for `action = "restart"`). Defaults to `false`.
- `triggers` (List of String) Arbitrary list of strings. When any value changes, the action is re-executed. Use `timestamp()` to force a re-run on every apply.

### Read-Only

- `created_at` (String) Timestamp of when the action was last executed.
- `id` (String) Opaque identifier for this action invocation. Always set to the node UUID.
